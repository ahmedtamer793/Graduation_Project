# 📌 Project Overview

This project demonstrates the complete Software Testing Life Cycle (STLC) by applying different testing techniques across four development sprints.

The project focuses on ensuring software quality through:

- Manual Testing
- Database Testing
- API Testing
- Automation Testing

Each sprint represents a different phase of the testing process and includes planning, execution, reporting, and validation.

### Objectives

- Understand Business Requirements

- Execute Test Cases
- Report Defects
- Verify Bug Fixes

- Test Plan
- Test Scenarios
- Test Cases
- Bug Reports
- Test Summary Report
